# Quick Start Guide

## Prerequisites

- Node.js 18+
- pnpm package manager

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

The application runs at `http://localhost:3000`.

## Project Structure

```
src/
├── app/          # Next.js app router pages
├── components/   # React components
└── lib/          # Utilities and shared code

workspace/        # Project data storage
```
