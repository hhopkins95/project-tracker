# Project Tracker Documentation

Welcome to the Project Tracker documentation. This is a Next.js application for managing projects, initiatives, todos, and ideas.

## Quick Links

- [Getting Started](./getting-started/quickstart.md) - Set up and run the project
- [Guides](./guides/) - Task-oriented guides
- [Reference](./reference/) - Technical reference and API docs

## Project Overview

Project Tracker helps you organize work through:

- **Initiatives** - Major project milestones or features
- **Todos** - Actionable tasks tied to initiatives
- **Ideas** - Captured thoughts for future consideration
- **Sessions** - Track work sessions and decisions
